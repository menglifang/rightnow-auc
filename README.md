## 描述

`rightnow-auc`将提供一个统一的用户身份认证服务，并提供多应用的单点登录（SSO）。其基于[devise](https://github.com/plataformatec/devise) 和[doorkeeper](https://github.com/applicake/doorkeeper) 进行开发。

*申明* 目前`rightnow-auc`还处于早期开发阶段。
